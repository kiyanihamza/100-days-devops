# Projet Day 96 - DevOps

Ce dépôt contient le code et la configuration liés au challenge DevOps du jour 96.

## Contenu

- `main.tf` : fichier Terraform principal pour provisionner l'infrastructure.

## Utilisation

1. Ouvrir le projet dans un éditeur.
2. Vérifier le contenu de `main.tf`.
3. Exécuter les commandes Terraform habituelles :
   - `terraform init`
   - `terraform plan`
   - `terraform apply`

## Remarque

Ce README est volontairement simple et vise à documenter rapidement l'objectif du projet.
